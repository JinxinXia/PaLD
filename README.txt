This is the updated version of method PaLD from Berenhaut, K. S., Moore, K. E. and Melvin, R. L.** (2021) A social perspective on perceived distances reveals deep community structure.

The original algorithm has an order of O({n^3}/2). We beat it using sorted algorithm with O(n^2 + \sum{|u_xy|}), where \sum{|u_xy|} is the sum of each size of local conflict focus.
